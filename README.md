# supermall
本项目使用vue-cli3构建购物商城平台，通过vue-router实现首页、详情页、购物车和个人信息页面间的跳转；采用了ES6的语法，各页面所需数据统一使用axios进行处理；对于购物车的数据状态采用vuex进行管理；此外用vue-lazyload的懒加载方式来展示首页商品、详情页等的信息；执行方法类似的代码块提取到mixin中进行复用。
